# Metadata1

Metadata1 is a library that helps you with amazon metadata1 encryption

## Installation

```bash
$ npm install metadata1
```

```bash
$ yarn add metadata1
```

## Usage

```typescript
import Metadata1 from 'metadata1';

Metadata1.encrypt(data: any): string; 
```
